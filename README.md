# Smart Climate Pro (Tado-like for Home Assistant)

Advanced predictive and learning climate controller for Home Assistant.

## Features
- **Predictive Pre-heating**: Learns room thermal rates and starts heating based on travel distance.
- **Global & Local Settings**: Centralized home configuration with per-room overrides.
- **Robust Safety**: Hardware failure detection, frost protection, and manual override timeouts.
- **Multi-valve Support**: Manage multiple TRVs in a single room with one master sensor.
- **Geofencing**: Dynamic distance-based targets (Away/Eco/Home).

## Installation
Copy  to your Home Assistant  folder and restart.
